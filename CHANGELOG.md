## [1.0.0] - 2026-07-19

首个稳定功能基线。

### 新增

- 支持 TXT、Markdown 和文本型 PDF 导入，保留页码、行号与原文映射；
- 支持常见中英文场景标题的确定性切分，以及经本地安全校验的模型辅助格式识别；
- 支持逐场叙事分析、三幕结构、节拍、主支线、人物关系、人物弧光、主题、母题和证据引用；
- 为全部已归一人物生成分级档案，并为核心人物生成区分观察、转述、推断和未知项的声明级小传；
- 支持叙事语义风险筛查、专家抽检、指纹绑定的人工修正、复核和稳定版门禁；
- 支持地点、演员、群演、服化道、车辆、动物、动作、特效、声音和复杂度等制片元素拆解；
- 支持拍摄单元、跨场资源实体与出现项、数量事实、单位语义和最大同时需求；
- 支持高危元素的确定性识别、专业角色审批范围和不可绕过的危险默认实现否决；
- 支持制片专家评审、原子化修正、不可变 generation，以及评测版和专业版分级封版；
- 提供 JSON、Markdown 和 CSV 导出，以及统一的 JSON CLI 输出与退出码；
- 提供内容指纹缓存、过期检测、逐场/逐人物恢复和可审计产物；
- 提供 Typer CLI、Pydantic 结构化边界、Agno/DeepSeek 适配和本地诊断命令。

### 安全与质量

- API Key 仅从环境读取，不写入项目产物、缓存、导出或日志；
- 模型生成的正则表达式在执行前经过长度、锚定、高风险语法和样本覆盖校验；
- 人工答案、修正、评审和安全批准均绑定内容或范围指纹，过期结果不会静默复用；
- 专业封版要求真人专家和逐风险专业角色批准，不把 AI 评测结果表述为现实开拍许可；
- 使用 Ruff、pytest、覆盖率与文件行数约束维护代码质量；
- 以 MIT License 开源，并提供贡献、安全、隐私、支持与社区行为准则；
- 使用 Python 3.12/3.13 CI、Dependabot 和依赖审查维护回归与供应链基线；
- 使用隔离构建、安装冒烟和 PyPI/TestPyPI Trusted Publishing 完成发行门禁。
